Bienvenue sur la page de SCNMPScanner

Nous vous présentons un script réseau basé sur Nmap pour scanner les connexion TCP.

Avant de commencer

vous devez tout d'abord installer Python-Nmap disponible sur: http://xael.ord/norman/python/python-nmap/

Comment l'utiliser

Sur un terminal ou un prompt shel cmd.exe, exécutez le script comme suit:

syscore$ python SCNMPScanner.py -H 192.168.1.137 -p 21, 22, 80, 1720
